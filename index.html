This is a Browser-based Application Environment for 3P

This Application examines an array of metrics that characterize the performance of Freeosk Kiosks

This Includes: Dispense/Stage Times, Errors, Utilization, Efficiency and more

This application was created to allow users to examine program performance as they see fit

The intent being the dynamic nature of this platform will allow us to answer questions, both broad and defined, quickly and accurately depending on the user's needs

Please note that when you visit internal pages, the application is performing analysis to prepare visualizations - The time to complete this process varies by page

Please allow 3P time to extract and refine data for consumption

Refreshing a page will re-run that page's analytical tools

At this time, all analysis is bound by current placements - please see the "Placements" Page for more info

As a note, this program is built using a JS/CSS/HTML front end with a Python-based Flask App and Custom Built Python Analysis Functions (3P's beginnings)
